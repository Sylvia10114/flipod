信息密度评分（绿/黄/红 = 极简/中/密）

  • ① OB 介绍关键节点屏：6 元素（音波 + 进度条 + 文字 + 2 chip）· 轻 · ✓ chip 让用户掌控节奏
  • ② Priming 普通态：8 元素 · 轻 · ✓ 中央倒数 ring 是视觉锚点，其他元素围绕它
  • ③ Phase 1 第一帧：13 元素 · 中 · ⚠ 字幕外的功能 chrome 较多
  • ④ 特殊 priming：12 元素 · 中-高 · ⚠ 智能体气泡 + 倒数 ring 同屏，注意力分散
  • ⑤ 侧边菜单：6 项 + header · 轻 · ✓ 标准 drawer pattern，用户熟悉

核心流程（用户视角）

  • 首次：OB 5 段介绍 → priming → phase 1-5 → priming（智能体介入） → phase 1-5 → priming → ...
  • 日常：priming → phase 1-5 → priming → phase 1-5 → ... 无限流
  • 侧边菜单随时可呼出 · 收藏 / 爱心 / 看看有什么 / 生词本 / 学习兴趣 都在这里

值得讨论的设计决策

  • ④ 智能体气泡出现时，倒数 ring 是否应暂停？（避免用户处理气泡时被推进 clip）
  • ③ Phase 1 第一帧的 phase label / title / keywords 是否在第一句字幕开始播后淡出，让沉浸氛围更纯？
  • ① OB 段落 ③（5 阶段流程）讲完出 chip，其他段落自动播 —— 是否所有段落都给 chip 让用户更有控制感？还是保持只在关键节点出 chip？
  • ⑤ 菜单项「学习兴趣」是否需要小红点提示首次去配置？

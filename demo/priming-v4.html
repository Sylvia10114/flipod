核心架构

  • 打开 app 直接就是 priming，不是 splash，不是 list
  • Priming 是 feed 的内容单元：上滑 = 下一条 priming
  • 点倒计时环 = 进入这条的 clip
  • 整体是 priming+clip 的竖滑无限流

Priming 交互

  • 倒计时：≤2 词→3s / 3 词→4s / 4-5 词→5s，rAF 驱动
  • 长按暂停：任意处按住 ≥160ms，环变灰，hold-tip 切换"松开继续"；松开 250ms 内吞 click 避免误跳
  • 竖滑换条：|dy| > min(90, H·0.18) 或速度 > 0.35 px/ms；首尾橡皮筋 ×0.35；横向手势优先让长按
  • 点环进 clip：任意剩余时间点环立刻进入；倒计时 0 也自动进

三层信息

  • Meta：source · CEFR · duration（11px，弱色）
  • Summary：中文"听力锚"≤16 字单行，不剧透结论
  • Keywords：EN 44 / ZH 14，4-5 词自动 dense（EN 32）

定位锁"轻预热"

  • 关键词不可点击（不展开例句/词根/发音）
  • priming 内不做收藏（收藏留给 clip）
  • 不暴露"一共 N 条 / 今天几条 / 还剩几条"
  • 不做 dots / skip-once 按钮 / 周报式进度

Settings

  • 单开关"默认显示 priming"，默认 ON
  • ON：home = priming，clip 有 back 按钮
  • OFF：home = clip player，上滑换下一条 clip，back 隐藏
  • 切换后 bootHome() 从第一条重新开始

键盘映射（桌面）

  • ↑ / ↓ 换条（priming 或 player-as-home）
  • space 按住 = 长按暂停
  • esc 返回（settings / clip）
  • Enter 立刻进 clip（priming 模式）

# Batch 3A: 产品全路径 UI 流程图

## 背景

Flipod 是一个 AI-native 英语听力 app，有两个核心模式：Feed（泛听）和 Practice（精听）。需要一张完整的产品路径图，展示用户可触达的所有路径和状态。

## 需求

创建一个交互式的 HTML 文件（单文件，用 React + Tailwind），展示 Flipod 的完整产品路径。

### 需要覆盖的所有路径

```
Onboarding:
  Screen 1 (CEFR 选择) → Screen 2 (兴趣标签) → Loading → Feed

Feed 主流程:
  上下滑动浏览 clip → 
    ├── 点击播放/暂停
    ├── 点词 → 弹出释义 popup → [收藏词汇] / [标记认识] / 关闭
    ├── 点击中文遮罩 → 显示翻译
    ├── 句子前进/后退导航
    ├── 点击爱心 → 记录偏好 → 影响推荐
    ├── 点击书签 → 收藏 clip → [首次触发 toast 引导]
    ├── AI 推荐理由（每个 clip 顶部显示）
    └── 功能卡片（穿插在 clip 之间）
         ├── Progress Card（听了几个、多长时间、查了几个词）
         ├── Review Card（间隔重复，复习收藏的词）
         └── Recommendation Card（检测偏好集中，建议调整）

侧边栏:
  汉堡菜单 → 
    ├── 我的收藏 → 收藏的 clip 列表 → 点击跳到对应 clip
    ├── 词汇本 → 统计卡 + 词汇列表 → 点击弹 popup
    ├── 听力练习 → Practice 入口
    │    ├── 你收藏的（未练习 / 已练习）
    │    └── AI 推荐练习（基于水平+词汇+兴趣）
    └── 设置
         ├── 深色/浅色主题切换
         └── 左手/右手模式切换

Practice 流程（从侧边栏进入）:
  选择 clip → 
    Step 1 盲听 → 
      ├── "大部分听懂了" → Step 4
      └── "有些没听清" → Step 2
    Step 2 逐句精听 → 
      每句: 播放 → 显示字幕 → [点词查释义] → 标记 ✓容易 / ✗困难 → 下一句
      → Step 3（如果有标记困难的句子）或 Step 4
    Step 3 难句闪卡 → 
      正面: 音频 + 英文 → 翻面: 中文 + 释义
      → "搞懂了" 移除 / "还不清楚" 排到最后
      → 全部完成 → Step 4
    Step 4 复听 + 总结 → 
      完整播放（带精听标记）→ 统计面板
      → "回到 Feed" / "再练一段"

数据闭环（用箭头标注）:
  Feed 点词 → 词进入词汇本 → Review Card 复习
  Feed 收藏 → clip 进入 Practice 列表
  Practice 标记难句 → 难词标记为精听过
  用户行为 → AI 调整 Feed 推荐排序
  听了 10 个 clip → Level 校准建议
```

### 视觉要求

- 用流程图/节点图的方式，不是线性列表
- 每个节点可点击展开详情（显示该页面的简要描述 + 截图占位框）
- 用不同颜色区分：Feed 路径（蓝）、Practice 路径（紫）、数据闭环（橙色虚线箭头）
- 深色背景，和 Flipod 的视觉风格一致
- 移动端友好（可缩放/滚动）

### 输出

单个 HTML 文件，保存到项目根目录的 `design/product-flow-map.html`
